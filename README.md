# make_images_Fantasy-Map-Generator

## ファンタジーマップ画像生成スクリプト(make_images_Fantasy-Map-Generator)の使い方  
  
    git clone https://github.com/yoramu/make_images_Fantasy-Map-Generator.git  
    cd make_images_Fantasy-Map-Generator  
    python drive.py  
        
ブラウザが開きseleniumが自動で30秒ほどで設定を行います。  
drive.pyがあるディレクトリーに日付の名前がついたフォルダーを作成しその中にpngとjsonとhtmlのデータを保存し始めます。  
複数起動可能ですが重くなってしまって逆に効率が悪くなることもあるので自分のPCのメモリと要相談  
    
注意事項：  
スリープ状態になったりブラウザを画面上にとらえておかないとエラー停止してしまいます。  
フォルダー内にあるchromedriveはOS環境によって変わるのでchromedriveをwebからダウンロードしてきてください。
